# 🏗️ Proyecto: Tienda de Zapatos Online  
## 🚀 Plan de Desarrollo en 6 Sprints

---

## 🎯 Objetivo del Proyecto
Construir una tienda online moderna, escalable y profesional para la venta de zapatos, iniciando desde una base frontend sólida y evolucionando hacia una aplicación completa conectada a Supabase.

---

## 🧩 🔹 SPRINT 1 — Base del Proyecto

### 🎯 Objetivo
Construir la base estructural y visual del proyecto.

### 🧠 Aprendizaje
- Estructura de proyectos
- Componentes
- Layouts
- Buenas prácticas

### 📦 Entregables
- Proyecto inicial en React / Next.js
- Estructura de carpetas organizada
- Layout principal (Header + Footer)
- Navegación funcional

### 🔧 Funcionalidades
- Navbar
- Footer
- Home base
- Diseño con Tailwind

### 📁 Estructura
components/
layouts/
pages/
styles/

### ✅ Criterio de éxito
La aplicación tiene estructura profesional y navegación funcional.

---

## 🧩 🔹 SPRINT 2 — Interfaz de Tienda

### 🎯 Objetivo
Construir la experiencia visual del ecommerce.

### 🧠 Aprendizaje
- Render dinámico
- Diseño UX/UI
- Componentes reutilizables

### 📦 Entregables
- Página principal completa
- Catálogo de productos
- Tarjetas de producto
- Vista de detalle

### 🔧 Funcionalidades
- Listado de productos (mock)
- Filtros visuales
- Vista de detalle
- Productos relacionados

### 🧠 Datos
Uso de JSON simulado

### ✅ Criterio de éxito
La tienda es visualmente funcional y profesional.

---

## 🧩 🔹 SPRINT 3 — Estado y Lógica Local

### 🎯 Objetivo
Agregar comportamiento real a la aplicación.

### 🧠 Aprendizaje
- Manejo de estado
- Lógica de negocio
- Separación de responsabilidades

### 📦 Entregables
- Carrito funcional
- Sistema de favoritos
- Gestión de cantidades

### 🔧 Funcionalidades
- Agregar al carrito
- Eliminar productos
- Actualizar cantidades
- Wishlist

### 🧠 Tecnologías
- useState
- Context API

### 📁 Estructura
hooks/
services/
utils/

### ✅ Criterio de éxito

---

## 🧩 🔹 SPRINT 4 — Integración con Supabase

### 🎯 Objetivo
Conectar el frontend con backend real.

### 🧠 Aprendizaje
- APIs
- Bases de datos
- Autenticación

### 📦 Entregables
- Conexión a Supabase
- Productos desde base de datos
- Login y registro

### 🔧 Funcionalidades
- Lectura de productos
- Autenticación de usuario
- Manejo de sesión

### 🗄️ Base de datos
- products
- categories
- brands

### 🔐 Seguridad
- Row Level Security (RLS)

### ✅ Criterio de éxito
Datos reales y autenticación funcionando.
